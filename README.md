# Playwright Smoke Tests – Careers Flow

This project contains Playwright automated smoke tests for the careers flow.

## Tech Stack

- Playwright
- JavaScript
- Page Object Model (POM)
- Faker (test data generation)

## Test Coverage

- Navigate to Careers page
- Open job search
- Open job details
- Start job application
- Fill application form (without submission)

## Run tests

Install dependencies:

- npm install

Run tests:

- npx playwright test
